# Bank_account
This is a Python program i have written, its a simple bank account system that lets the user deposit or withdraw money. It updates and displays the balance after each transaction.

## Built With
- Python
- Object-Oriented Programming
- User Input
- Conditional Statements

## Features
- Create a bank account with a starting balance
- Deposit money into an account
- Withdraw money from an account
- Display updated account balances
- Interactive command-line banking system
- Basic account balance validation
